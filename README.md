# Panoverlay

Augmentations for PTGui to assist in stitching difficult panoramas.

![2026-04-02_17-19-32](https://github.com/user-attachments/assets/b77660ce-cc69-4659-aa8a-eeef7255e160)

## Features:

- Draw lines in a transparent overlay (meant to be positioned on top of PTGui's Panorama Editor) which visualize control point quality and image relationships.
- Toggle between absolute and relative distances for line weighting.
